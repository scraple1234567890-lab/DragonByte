Detailed breakdown

What’s included, what to prepare, and typical outcomes for each service.

1) Home Wi-Fi setup/optimization

Fix dead zones, improve stability, and reduce “it works sometimes” behavior.

What’s included

  • Wi-Fi signal review + coverage mapping
  • Router / mesh placement recommendations
  • Channel, band, and basic security tuning
  • Device inventory snapshot (key devices)
  • Validation tests for speed, latency, and stability

What to prepare

  • ISP name (optional)
  • Router model (optional)
  • Approx. number of devices
  • Where issues happen (rooms/areas)

Typical outcomes

  • Fewer drops on calls/streams
  • Better coverage in problem areas
  • Cleaner settings you can understand
  • Simple handoff notes for future reference

2) Small office network setup/optimization

Reliable networking for printers, POS, VoIP, shared drives, and day-to-day work.

What’s included

  • Network layout review (router/switch/APs)
  • Basic segmentation as needed (guest vs business)
  • Wi-Fi performance improvements (roaming, stability)
  • Critical device validation: POS, printers, VoIP, shared drives
  • Labeling and cleanup recommendations

What to prepare

  • Number of employees/devices
  • List of key apps/tools (email, VoIP, POS, drives)
  • Any known “always breaks” workflows
  • Admin contact who owns/administers network

Typical outcomes

  • More consistent connectivity
  • Fewer “printer disappeared” issues
  • Cleaner separation between guest and business devices
  • Documentation that reduces dependency on one person

3) Network design + documentation (“Network Binder”)

Design clarity and a binder that makes troubleshooting and onboarding sane.

What’s included

  • High-level network diagram (simple and readable)
  • Device inventory (model, role, location)
  • Wi-Fi names/SSIDs (as appropriate) and usage guidance
  • Support contacts and vendor list
  • Where access info lives (without exposing secrets)

What to prepare

  • Access to view network equipment settings
  • List of vendors (ISP, VoIP, POS, etc.)
  • Device locations (or photos of the setup area)

Typical outcomes

  • Less downtime from “unknown setup” problems
  • Faster support calls (you have the facts ready)
  • Cleaner handoff when staff changes
See deliverables

4) Custom employee training packs (“Company Network Playbook”)

Practical guides that match how your business actually operates.

What’s included

  • Role-based quick guides (front desk, manager, etc.)
  • Common tasks: Wi-Fi, printers, drives, email, VoIP
  • “If this happens, do this” checklists
  • Simple escalation paths and who to contact

What to prepare

  • List of tools (Google/M365, POS, VoIP, drives)
  • Common mistakes or frequent questions
  • Any policies you want included (optional)

Typical outcomes

  • Faster onboarding
  • Fewer interruptions for “basic” questions
  • More consistent handling of recurring issues
See packages

5) Tech Outage Recovery Plan (for their specific tools)

A step-by-step “What to do if X goes down” plan for the tools you rely on.

What’s included

  • Priority order (what to check first)
  • Who to call for each scenario
  • Where access info lives
  • Fallback options (hotspot, spare device, paper process)
  • Plans for: ISP/internet, router, Wi-Fi, POS, printers, email (Google/M365), VoIP phones, shared drives, key SaaS apps

What to prepare

  • Vendor contact info (ISP, POS, VoIP, etc.)
  • List of critical workflows and hours of operation
  • Spare device availability (optional)

Typical outcomes

  • Less panic, faster recovery
  • Staff follows the same steps consistently
  • Reduced downtime during outages
See scenarios

  • Basic issue description (where/when it happens)
  • Approx. number of devices
  • Router/mesh model (optional)
  • ISP name (optional)

For Business

  • Number of employees/devices
  • Tool list (POS, VoIP, email, drives, key SaaS)
  • Vendor contacts (if available)
  • Confirmation you own/administer the network
